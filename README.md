# Ecommerce App Backend

## Overview
Ecommerce App Backend is a backend application developed to manage core functionalities of an online shopping platform. The project provides secure REST APIs for user authentication, product management, cart operations, and order handling.

---

## Features

### User Authentication & Authorization
- User Registration and Login
- JWT-based Authentication
- Role-based Access Control
- Secure Password Handling

### Product Management
- Add Products
- Update Product Details
- Delete Products
- View All Products
- Category-wise Product Handling

### Shopping Cart Functionality
- Add Items to Cart
- Remove Items from Cart
- Update Product Quantity
- View Cart Details

### Order Management
- Place Orders
- View Order History
- Manage Order Status

### User Management
- User CRUD Operations
- Profile Management
- Protected Routes

### Database Integration
- Persistent Data Storage
- Database Connectivity
- ORM-based Operations

### REST API Development
- RESTful API Architecture
- JSON Request & Response Handling
- API Testing using Postman

---

## Technologies Used
- Golang
- Gorilla Mux
- JWT Authentication
- GORM
- MySQL
- REST APIs
- Postman

---

## Project Purpose
The application is designed to simulate the backend architecture of a real-world e-commerce platform by handling user management, product inventory, shopping cart operations, and order processing efficiently.
